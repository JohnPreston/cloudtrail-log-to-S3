eucalyptus-cloudtrail
=====================

Simple ansible role that can be used to log all API calls in S3

Requirements
------------

S3 credentials

Role Variables
--------------

s3_url


Example Playbook
----------------

```

- hosts:
  - clc
  roles:
  - eucalyptus-cloudtrail

```
